Interface RuleComponent

Validation rules define the validation on the editor fields.

Each field will create an instance of the validation rule based on the validation definitions in the field config.

isRequired

isRequired: boolean
  • Defined in selective/validationRules.ts:60

Does the rule make the value required?

The UI needs to be able to mark fields that are required without needing to show the entire error state.

If a validation rule makes it required, this allows the UI to flag the field for the user without a full error display.
